# 待办事项应用 - To-Do List App

一个功能完整的待办事项管理应用，支持本地存储、数据导入导出、任务过滤等功能。

## ✨ 功能特性

### 核心功能
- ✅ **添加待办项** - 快速添加新任务，支持回车快捷键
- ✅ **标记完成** - 勾选复选框标记任务完成
- ✅ **删除任务** - 删除单个或批量删除任务
- ✅ **任务过滤** - 按全部/进行中/已完成分类显示
- ✅ **优先级设置** - 支持高/中/低三个优先级

### 存储与导出
- ✅ **本地存储** - 使用 LocalStorage 自动保存数据
- ✅ **数据导出** - 导出为 JSON 格式文件
- ✅ **数据导入** - 从 JSON 文件导入任务
- ✅ **存储检测** - 自动检测本地存储可用性

### 统计与展示
- ✅ **实时统计** - 显示总数、活动数、完成数
- ✅ **完成率** - 实时计算任务完成率
- ✅ **时间戳** - 记录任务创建时间
- ✅ **空状态提示** - 无任务时显示友好提示

### 用户体验
- ✅ **响应式设计** - 完美适配手机、平板、桌面
- ✅ **流畅动画** - 使用 CSS 动画增强体验
- ✅ **通知提示** - 操作反馈提示
- ✅ **字符计数** - 实时显示输入字符数

## 🎯 技术栈

- **HTML5** - 语义化标记
- **CSS3** - 渐变、弹性布局、动画
- **Vanilla JavaScript** - 原生 JS 实现，无框架依赖

- **File API** - 文件导入导出功能

## 📖 使用指南

### 添加待办项
1. 在输入框中输入任务描述
2. 点击"添加"按钮或按回车键
3. 任务将添加到列表顶部

### 管理待办项
- **标记完成**: 勾选左边的复选框
- **删除任务**: 点击右边的"✕"按钮
- **查看时间**: 悬停查看任务创建时间

### 过滤任务
- **全部**: 显示所有任务及其计数
- **进行中**: 只显示未完成的任务
- **已完成**: 只显示已完成的任务

### 批量操作
- **清空已完成**: 删除所有已完成的任务
- **全部清空**: 删除所有任务（需要确认）

### 数据管理
- **导出**: 将所有任务导出为 JSON 文件，可备份或转移
- **导入**: 从 JSON 文件导入任务，会覆盖现有数据

## 💾 本地存储

### 存储位置
- **存储键**: `todolist_v1`
- **存储方式**: Browser LocalStorage
- **容量**: 通常 5-10MB（取决于浏览器）

### 存储内容
```json
[
  {
    "id": 1717829000000,
    "text": "完成项目报告",
    "completed": false,
    "priority": "high",
    "createdAt": "2024-06-09 09:30:00"
  }
]
```

### 自动保存
- 每次添加、删除、完成操作都会自动保存
- 页面刷新后数据自动恢复
- 支持浏览器 DevTools 查看存储数据

## 📊 数据结构

```javascript
// 单个待办项对象
{
  id: Number,              // 唯一标识（时间戳）
```

## 🐛 常见问题

### Q: 数据在哪里保存？
A: 数据保存在浏览器的 LocalStorage 中，不会上传到服务器。不同浏览器/设备的数据隔离。

### Q: 删除后能恢复吗？
A: 不能。建议经常导出备份。可以考虑添加撤销功能。

### Q: 为什么导入后原数据消失了？
A: 导入会覆盖现有数据。请先导出备份再导入。

### Q: 支持云同步吗？
A: 当前版本不支持。可以使用导出/导入功能实现跨设备数据转移。

### Q: 能在手机上使用吗？
A: 可以。应用完全响应式，支持所有现代浏览器。
